## 一、 HTML 基础知识

### 1\. 表格单元格合并

* **跨行合并 (rowspan)**：合并上下单元格，例如 `<td rowspan="2">`。
* **跨列合并 (colspan)**：合并左右单元格。
* *注意*：合并不能跨越表格的结构标签（`thead`, `tbody`, `tfoot`）。

### 2\. 表单元素详解

* **`<input>` 输入框及属性**：

  * `type="text"`：普通文本框。
  * `type="password"`：密码框（输入内容不可见）。
  * `type="radio"`：单选框。需配合 `name` 属性分组实现单选，`checked` 设置默认选中。
  * `type="checkbox"`：多选框。
  * `type="file"`：文件上传。添加 `multiple` 属性可上传多个文件。
  * `placeholder` 属性：占位文本提示。
* **下拉菜单**：使用 `<select>` 和 `<option>`（加 `selected` 可默认选中）。
* **多行文本域**：`<textarea>`。可通过 `cols` 和 `rows` 控制宽高，能自动换行。
* **`<label>` 标签**：用于扩大表单控件的点击范围（例如点击文字就能勾选复选框）。
* **按钮 `<button>`**：

  * `type="submit"`：提交数据到后台（默认功能）。
  * `type="reset"`：重置表单。
  * `type="button"`：无默认功能，通常配合 JS 使用。

## 二、 CSS 基础入门 

### 1\. CSS 简介与引入方式

CSS（层叠样式表）用于描述 HTML 的呈现观感。

* **内部样式表**：写在 HTML `<head>` 内的 `<style>` 标签中（适合学习测试用）。
* **外部样式表**：写在独立的 `.css` 文件中，通过 `<link rel="stylesheet" href="...">` 引入（实际开发推荐）。
* **行内样式**：直接写在标签内部，通常配合 JavaScript 动态修改时使用。

### 2\. 基础选择器

选择器的作用是“查找标签，并设置样式”。

* **① 标签选择器**：以标签名命名（如 `p`, `div`, `a`）。会选中所有相同的标签，无法差异化。
* **② 类选择器 (`.`)**：定义时用 `.` 开头，标签内通过 `class="类名"` 调用。最常用，可差异化设置样式，且一个标签可使用多个类名。
* **③ id 选择器 (`#`)**：定义时用 `#` 开头，标签内通过 `id="id名"` 调用。一般配合 JS 使用，较少用于纯 CSS。
* **④ 通配符选择器 (`\*`)**：查找页面所有标签并设置相同格式，无需调用直接生效。
